# Habit Tracker

A clean, minimal habit tracker web app to build and maintain daily habits. Track your progress with streak counters and a weekly overview chart — all stored locally in your browser.

## Features

- **Custom Habits** — Add unlimited habits with a simple input form
- **Daily Check-off** — Mark habits as complete each day with a single click
- **Streak Counter** — Automatically tracks consecutive days for each habit
- **Weekly Chart** — Visual bar chart showing completion across the last 7 days
- **Dark / Light Mode** — Toggle between themes; preference is saved
- **LocalStorage** — All data persists in your browser with no account needed
- **Responsive Design** — Works on desktop, tablet, and mobile

## How to Use

1. **Open** `index.html` in any modern browser
2. **Add a habit** by typing in the input field and clicking "Add"
3. **Check off** habits as you complete them throughout the day
4. **Track streaks** displayed next to each habit
5. **Review** your weekly progress in the chart below
6. **Switch themes** using the sun/moon button in the header

No build tools, frameworks, or server required — just open and use.

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- Canvas API for the weekly chart
- localStorage for data persistence
